Найдите в интернете информацию про класс java.util.Optional.
1. Для чего применяются объекты этого класса?
2. Какие два состояния могут быть у объектов этого класса?
3. Как создаются экземпляры данного класса?
4. Как можно проверить в каком состоянии находится экземпляр данного класса?
5. Как можно получить объект, который находится внутри Optional?

1. Объекты этого класса служат футляром для объекта, тем самым помогают избежать NullPointerException.
2. Футляр может иметь объект внутри, либо быть пустым.
3. Нужно указать тип объекта Optional и указать имя этого экземпляры => Optional name;
4. Метод isPresent() проверяет наличие обхекта в футляре. Если объект есть, то метод выдаст true, и false
в обратном случае.
5. С помощью метода getAs(), мы можем получить объект, который находится в футляре
